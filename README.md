# 🖥️ Windows 95 Desktop Simulator

A fully interactive Windows 95 desktop simulator running in the browser — complete with boot sequence, draggable windows, start menu, taskbar and more!

## 👤 Author
**Tanzim Hossain Ayon** | CSE Graduate · BRAC University · IEEE Researcher
🌐 [Portfolio](https://tanzimhossainayon.vercel.app)

## 🚀 Features
- 💾 Animated boot sequence with progress bar
- 🖥️ Full desktop with icons
- 🪟 Draggable, minimizable, maximizable windows
- 📋 Start menu with navigation
- ⏰ Live taskbar clock
- 👤 About Me window with personal info
- 💻 Skills window with progress bars
- 🔬 Research/IEEE Publication window
- 📧 Contact window with links
- 📝 Working Notepad
- 🌐 Internet Explorer window
- 🗑️ Recycle Bin
- 💾 Window positions saved via LocalStorage
- 🎨 Canvas wallpaper background

## 🛠️ Technologies Used
| Technology | Purpose |
|---|---|
| HTML5 | Structure & layout |
| CSS3 | Windows 95 UI styling |
| JavaScript | Window management & interactions |
| Canvas API | Wallpaper rendering |
| LocalStorage | Save window positions |
| CSS Variables | Theme management |
| DOM Events | Drag, click, keyboard |
| VT323 Font | Retro pixel font |

## ▶️ How to Run
Open `index.html` in any browser — no installation needed!
